# 表格列宽度修改总结

## 修改时间
2025年

## 修改目标
将所有表格页面的固定像素宽度改为百分比宽度，确保：
1. 表格容器宽度为 100%
2. 各列宽度使用百分比（min-width）
3. 表格横向铺满整个卡片区域
4. 每列宽度按内容比例分配
5. 阅读舒适且操作按钮不会换行
6. 如果屏幕过窄，允许水平滚动

## 修改范围
- 共修改 **68 个文件**
- 共修改 **951 列**的宽度设置

## 宽度映射规则
| 原固定宽度 | 新百分比宽度 | 适用场景 |
|-----------|-------------|---------|
| 55px | 5% | 选择框列 |
| 80px | 6% | ID列 |
| 100px | 8% | 较短列 |
| 120px | 10% | 中等列 |
| 150px | 12% | 较长列 |
| 180px | 14% | 很长列（如身份证） |
| 200px | 15% | 很长列 |
| 250px | 18% | 超长列 |

## 注意事项
- 所有表格已设置 `style="width: 100%"`
- 使用 `min-width` 而非 `width`，允许列根据内容自适应
- 保持了 `fixed` 属性（如有），用于固定列
- 保持了 `show-overflow-tooltip` 属性，防止内容溢出
