## 业务背景

本系统面向中大型企业（500~5000人），核心目标：
1. 实现无限层级部门树管理
2. 规范员工全生命周期管理（入职→在职→离职）
3. 支持员工调岗、晋升、调薪等异动记录
4. 智能化证件到期提醒，减少人工管控
